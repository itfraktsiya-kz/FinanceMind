📚 Инструкция по использованию FinanceMind

Добро пожаловать в FinanceMind!

Эта инструкция поможет вам быстро освоиться в приложении и начать удобно управлять своими финансами.

1. Главный экран

  • Показывает ваши расходы, средние траты и расходы за месяц
  • Есть «Совет дня» от ИИ-помощника
  • Можно создавать и отслеживать финансовые цели

2. Аналитика

  • График показывает изменения расходов по дням
  • Диаграмма показывает, на что уходят деньги по категориям
  • Можно просматривать отчёт за месяц и сохранять данные

3. Цели

  • Создавайте финансовые цели для накопления
  • Указывайте сумму и срок достижения цели
  • Следите за прогрессом накопления в реальном времени

4. Миссии (Геймификация)

  • Выполняйте задания для развития финансовых привычек
  • Получайте монеты FinCoin за выполнение миссий
  • Отслеживайте прогресс выполнения через проценты

5. Магазин

  • Тратьте FinCoin на бонусы и премиум-функции
  • Покупайте дополнительные функции для улучшения опыта
  • Активируйте премиум-подписку для расширенных возможностей

6. ИИ-ассистент

  • Отвечает на ваши вопросы по финансам
  • Помогает с использованием приложения
  • Доступен всегда для консультаций

Всё просто, понятно и доступно — начните управлять финансами уже сегодня!
